Client-Server Architecture

Diagram

    User (Client)
        |
        | Request (HTTP)
        ↓
    Web Server
        |
        | Response (HTML, CSS, JS)
        ↓
    Browser Displays Website

Explanation

Client-server architecture is a model where the client (user's browser) sends a request to the server, and the server processes the request and sends back a response.

Steps when opening a website

  1. User enters URL in browser
  2. Browser sends request to server
  3. Server processes request
  4. Server sends response (HTML, CSS, JS)
  5. Browser renders the webpage
